# Set-AzureRMTenantSubscription

## SYNOPSIS
Updates a tenant subscription.
This cmdlet is applicable only to Microsoft Azure Stack Environments

## SYNTAX

```
Set-AzureRMTenantSubscription -Subscription <SubscriptionDefinition> [-AdminUri <Uri>] [-Token <String>]
 [-ApiVersion <String>] [-DisableCertificateValidation] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```
